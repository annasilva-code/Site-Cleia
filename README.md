# Cleia Cabelos — Site simples

Como ver o site localmente:

1. Abra o arquivo `index.html` no navegador.

No macOS, no terminal, execute:

```bash
open index.html
```

2. Para editar, altere os arquivos em `index.html`, `css/styles.css` e `assets/logo.svg`.
